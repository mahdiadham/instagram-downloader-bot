# Instagram Bot
An Instagram Downloader Bot With Personal API

## Technologies :
<img src="https://skillicons.dev/icons?i=nodejs,express,typescript,mysql&perline=4" />

## Instructions
1. Clone the repository first and install the requirements packages :
   
   ```bash
   git clone https://github.com/mahdiadham/instagram-downloader-bot.git
   cd instagram-downloader-bot
   npm install
   ```
2. Create .env file in the root directory and replace with your own values :
   
   ```bash
   BOT_TOKEN=
   API_URL=http://localhost:8000/api/download/
   DB_USERNAME=
   DB_PASSWORD=
   DB_HOST=
   DB_PORT=
   DB_NAME=
   ```
3. Develope the bot with command :
   
   ```bash
   npm run dev
   ```
4. Build the project :
   
   ```bash
   npm run build
   ```
5. Launch the bot with command :
   
   ```bash
   npm start
   ```
6. Enjoy the bot !
