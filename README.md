# Instagram Bot
An Instagram Downloader Bot with NodeJS

## Technologies :
<img src="https://skillicons.dev/icons?i=ndoejs,typescript,mysql&perline=3" />

## Instructions
1. Clone the repository first and install the requirements packages :
   
   ```bash
   git clone https://github.com/mahdiadham/instagram-downloader-bot.git
   cd instagram-downloader-bot
   npm install
   ```
2. Create .env file in the root directory and replace the values :
   
   ```bash
   BOT_TOKEN=
   API_URL=
   DB_USERNAME=
   DB_PASSWORD=
   DB_HOST=
   DB_PORT=
   DB_NAME=
   ```
3. Launch the bot with command :
   
   ```bash
   npm start
   ```
4. Enjoy the bot !
